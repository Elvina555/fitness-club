# Fitness Club - курсовая работа

### Предварительные требования

- **Локальный веб-сервер** (рекомендуется OpenServer для Windows)
- **PHP** версии 8.2
- **MySQL** версии 8.0
- **Apache 2.4**
- **Git**
- **Расширения PHP**: mysqli, json, mbstring

### Установка и настройка

### 1. Установка OpenServer

1. Скачайте OpenServer с [официального сайта](https://ospanel.io)
2. Установите программу в удобную директорию (например, C:\OSPanel)
3. Запустите OpenServer через ярлык на рабочем столе
4. В трее (возле часов) найдите иконку OpenServer и нажмите **"Запустить"**

### 2. Настройка версий и модулей в OpenServer

1. В трее нажмите на иконку OpenServer **правой кнопкой мыши**
2. Выберите **"Настройки" → "Модули"**
3. Убедитесь, что выбраны следующие версии:
   - **Apache 2.4**
   - **PHP 8.2**
   - **MySQL 8.0**
4. Включите необходимые модули (например, Apache, MySQL, PHP CLI)
5. Нажмите **"Сохранить"** и **"Перезапустить"**

### 3. Подготовка проекта

1. Откройте панель OSPanel
2. Найдите ваш проект **fitness-and-tests** в списке доменов
3. Убедитесь, что для проекта указаны правильные настройки:
   - **Домен:** fitness-and-tests
   - **Используется PHP 8.2**
   - **HTTP сервер:** Apache

### 4. Размещение файлов проекта

1. Откройте проводник Windows
2. Перейдите в директорию OSPanel: C:\OSPanel\home\
3. Найдите папку fitness-and-tests
4. Скопируйте все файлы вашего курсового проекта в эту папку

**Важно:** Главная страница должна быть доступна по пути:  
C:\OSPanel\home\fitness-and-tests\main_index.php

### 5. Настройка базы данных

1. В панели OSPanel нажмите на проект **phpmyadmin**  
   или перейдите в браузере по адресу: http://phpmyadmin/
2. Авторизуйтесь (данные по умолчанию):
   - **Логин:** root
   - **Пароль:** оставьте пустым
3. Создайте новую базу данных для проекта
4. Импортируйте SQL-дамфайл проекта (fitness-club.sql)

**Примечание:** В проекте используется SQL-дампфайл (fitness-club.sql), содержащий структуру и начальные данные. Если вы используете не MySQL или phpMyAdmin, вы можете восстановить базу вручную, выполнив SQL-запросы из этого файла.
